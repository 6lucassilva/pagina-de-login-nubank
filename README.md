# pagina de login nubank
 Para testar o conhecimento que estou adquirindo em HTML e CSS resolvi fazer uma cópia da págna de login do Nubank. Não contente, ainda adicionei um pouquinho de JavaScript para deixar a interação com botão mais legal. Apesar de simples, esse foi um projeto bem desafiador, onde foi necessário aprender a importância dos espaçamentos e tamanhos dos box model.
